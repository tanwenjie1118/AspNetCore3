# This is an extension framework for Native AspNetCore 3.x

What we support now or in the future we will ?

- Mini Profiler
- IP Rate Limit
- Swagger developer document
- SignalR
- Docker CI
- Hosted Task scheduler
- Log4net/Nlog
- Redis
- MongoDB/SqlLite
- IdentityServer4
- Files service
- Email service
- Jwt token
